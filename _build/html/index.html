Learning Lean 4

Verus Institute
Arizona State University

These are my working notes and exercises while learning Lean 4, an interactive theorem prover and functional programming language 1. They follow along with An Introduction to Lean 4 — An Introduction to Formal Verification by Enric Cosme Llópez and Lü Gong 2 (course page).

The prose in this book is paired with runnable Lean source in the LearningLean/ library of the accompanying Lake project, so every idea here can be checked by the compiler.[1] Where the notes touch existing mathematics, they lean on Mathlib, Lean’s community mathematical library 3.

Chapters¶

  • Foundations — a knowledge graph of the skills that build toward formalizing mathematics in Lean.

  • Basic syntax — #check, #print, definitions, and lambdas.

  • Natural deduction — propositional proofs, including a Fitch-style derivation rendered from Typst.

  • Linear algebra — vectors, matrices, and linear maps, with Mathlib spellings (a work in progress).
